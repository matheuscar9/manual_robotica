# Manual de Cirurgia Robotica

Demo mobile criada em React Native com Expo para validar o fluxo inicial do aplicativo.

## O que ja existe

- Cadastro de usuario com nome completo, email e senha.
- Login local para a demo.
- Tela inicial com espaco reservado para a imagem de entrada.
- Menu com Inicio, Cirurgias, Admin e Ajustes.
- Selecao de cirurgia robotica.
- Tela com video, descricao e etapas do procedimento.
- Area de administrador para trocar a URL do video e a descricao.
- Preferencias de modo escuro e notificacoes.

## Como rodar

```bash
scripts\start-expo.cmd
```

No Android, instale o Expo Go pela Play Store e leia o QR Code exibido.

O Node.js LTS portatil fica em `.tools/`, entao nao precisa instalar Node globalmente no Windows.
